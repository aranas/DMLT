Getting Started

MLN Toolbox software implements various machine learning tools with an emphasis on their use in neuroscience. The toolbox provides a general interface to support the integration of new methods by writing high level wrappers. It allows complex methods to be built from simple building blocks and makes the use of cross-validation and permutation testing as easy as writing one line of Matlab code.
