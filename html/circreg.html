dml.circreg
 CIRCREG circular regression method class
 
    DESCRIPTION
    Circular regression using the Fisher and Lee model.
  
    EXAMPLE 
    
    X = randn(1000,2);
    c = dml.circreg('mu',0,'beta',[-1 1]','kappa',1);
    Y = c.sample(X);
 
    effect of mean regressors:
     c = dml.circreg('mu',0,'beta',[-1 1]','kappa',1);
     
     Y = c.sample(X);
     scatter(X(:,1),Y));
     hold on;
     scatter(X(:,2),Y,'r');
     xlim([-pi pi]);
     ylim([-pi pi]);
     d = c.train(X,Y);
     fplot(@(x)(d.beta(1)*x),[-pi pi],'b');
     fplot(@(x)(d.beta(2)*x),[-pi pi],'r');
     
    effect of concentration regressors:
     c = circreg('mu',0,'kappa',1,'verbose',true,'gamma',[1]');
     X = randn(1000,1);
     Y = c.sample(X);
     scatter(X,Y)
     xlim([-pi pi]);
     ylim([-pi pi]);

Constructor Summary
circreg CIRCREG circular regression method class 
Property Summary
mu mu - intercept of von Mises distribution 
kappa kappa - concentration parameter of von Mises distribution 
beta beta - regression coefficients for the mean 
gamma gamma - regression coefficients for the concentration 
mode mode - type of circular regression: none, mean, concentration, mixed 
lambda lambda - regularization parameter for the mean (only defined for gradient descent) 
verbose verbose - whether or not to generate diagnostic output 
restart when false, starts at the previously learned parameters  
Method Summary
  sample sample from a von Mises distribution (pp. 49 Fisher and Lee) 
  loglik log likelihood up to a constant 
  test estimate the angle using intercept and link function 
  train_mixed estimate mean and concentration using regressors 
  train_concentration estimate concentration using regressors 
  train_mean estimate mean using regressors 
  train_none don't regress; just estimate mean and concentration 
  train determine intercept and regression coefficients  
  model this method does not return a model
