dml.whiten
 WHITEN whitens the data
 
   DESCRIPTION
   Whitening of data through a whitening matrix W. Input data will be 
   standardized automatically.
 
    EXAMPLE
    X = rand(10,200);
    m = dml.whiten('indim',[2 100],'tdim',1);
    m = m.train(X);
    Z = m.test(X);
